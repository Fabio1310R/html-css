Testando cargas e imagens.

abaixo você vai ver a imagem que esta na mesma pasta.

podemos carregar imagens que estão em sub pastas tambem

tambem podemos carregar imagens externas

para adicionar uma imagem você pode fazer de três formas:
- usar uma img na mesma pasta
- usar uma img em uma sub pasta
- usar uma url

para todas as formas você deve criar uma tag img e colocar o nome ou url do arquivo em src

obs:Um tamanho bom para uma img em um site é de 600/650 pixels de largura,você pode redimensior a img utilizando o GIMP

Coloque o mouse entre as aspas e aprete ctrl + espaço para usar um atalho para imagens
